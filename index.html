02セレクタ（誰に）

CSSの「誰に」の部分。3パターン覚えれば足りる。

タグ名そのタグ全部
— CSS — h1 { color: red; }
↓ HTMLの <h1> 全部に効く
見出しが全部赤
.クラス名指定したやつだけ
— HTML — <p class="ピンク">これ</p> — CSS — .ピンク { color: deeppink; }
↓ .（ドット）忘れずに
これだけピンク
#ID名1個だけ特別
— HTML — <p id="主役">俺</p> — CSS — #主役 { color: gold; }
↓ #（シャープ）使う
俺だけ金

07classの使い方

「特定の1個だけ」装飾したいときの切り札。

HTML側 class="名前"  ＋  CSS側 .名前 { }
ポイント：CSS側にはドット . をつける。これがないと「そんなタグ無いよ」って無視される。

例：3つの段落のうち1つだけ青くする

CLASS で狙い撃ち
— HTML — <p>普通の文1</p> <p class="特別">俺だけ青</p> <p>普通の文2</p> — CSS — .特別 { color: blue; font-weight: bold; }

普通の文1

俺だけ青

普通の文2

例：同じclassを複数につけて、まとめて装飾

CLASS は使い回せる
— HTML — <p class="推し">春日</p> <p>誰か</p> <p class="推し">若林</p> — CSS — .推し { color: deeppink; font-size: 24px; }

春日

誰か

若林
